MyFlixAngularClient

Description

This project is a frontend build for a movie database and REST API that allows users to look up information about included movies, create a profile, and save favorite movies to their profile.

Link to GitHub Pages Site

myFlix-Angular

Technologies Used

  • Angular 16.0.0
  • Angular Material
  • RxJS
  • Typedoc

Set Up

  • Clone the repository
  • Run the command npm install from the root directory to install dependencies.
  • Run the command ng serve and navigate to http://localhost:4200/ in your browser.
